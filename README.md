# Enterprise Website

📌 Repo template cho khóa học **Git & DevOps**.

## Structure
- `src/` → Website chính (HTML/CSS/JS)
- `admin/` → Admin dashboard
- `blog/` → Blog/news
- `docs/` → Tài liệu dự án
- `docker/`, `ansible/`, `k8s/`, `ci-cd/` → Lab DevOps
- `monitoring/`, `security/` → Monitoring & Security
